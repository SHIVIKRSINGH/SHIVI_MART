📖 Format Guide

Required Fields:

  • name - Product name (text)
  • category_id - Category ID (number)
  • price - Price in rupees (number)
  • unit - Unit of measurement (kg, gram, piece, liter, dozen, bunch)
  • stock_quantity - Available stock (number)

Optional Fields:

  • description - Product description (text)
  • image_url - Main image URL (text)
  • image_url_2 - Second image URL (text)
  • image_url_3 - Third image URL (text)
  • discount_percentage - Discount % (number, 0-100)
  • is_available - Availability (1 or 0)
  • is_featured - Featured product (1 or 0)

Example CSV:

name,category_id,price,unit,stock_quantity,description,image_url,image_url_2,image_url_3,discount_percentage,is_available,is_featured Fresh Tomatoes,1,30,kg,100,Red ripe tomatoes,https://example.com/tomato1.jpg,https://example.com/tomato2.jpg,,10,1,0 Organic Apples,1,150,kg,50,Sweet red apples,https://example.com/apple1.jpg,,,5,1,1

Example JSON:

[ { "name": "Fresh Tomatoes", "category_id": 1, "price": 30, "unit": "kg", "stock_quantity": 100, "description": "Red ripe tomatoes", "image_url": "https://example.com/tomato1.jpg", "image_url_2": "https://example.com/tomato2.jpg", "image_url_3": "", "discount_percentage": 10, "is_available": 1, "is_featured": 0 } ]
