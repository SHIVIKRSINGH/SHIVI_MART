📖 Format Guide

Required Fields:

  • name - Product name (text)
  • category_id - Category ID (number)
  • price - Price in rupees (number)
  • unit - Unit: kg, gram, piece, liter, dozen, bunch
  • stock_quantity - Available stock (number)

Optional Fields:

  • description - Product description
  • image_url - Main image URL
  • image_url_2 - Second image URL
  • image_url_3 - Third image URL
  • discount_percentage - Discount % (0-100)
  • is_available - Availability (1 or 0)
  • is_featured - Featured product (1 or 0)
